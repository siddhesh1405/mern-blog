# MERN Blog Application

This is a full-stack Blog application built using the MERN stack as part of a hands-on learning project.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS

## Features
- RESTful API using Node.js and Express
- React frontend with routing
- MongoDB for data storage
- GET and POST requests
- API testing using Postman
